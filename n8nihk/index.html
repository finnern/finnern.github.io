⚡ 1. Den Kessel anfeuern: n8n & Docker

Wir nutzen Docker, um Windows-Sicherheitssperren zu umgehen[cite: 108].

  1. Windows + R drücken, cmd eingeben, Enter[cite: 109, 111].
  2. Befehl kopieren und in die CMD einfügen:
docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
Fehler "Conflict"? Erst docker rm -f n8n ausführen[cite: 115, 116].

Portal: http://localhost:5678 [cite: 117]

🧠 2. Lokale KI: Ollama & Phi-3.5

Datenhoheit: Alles läuft lokal auf Ihrem Rechner[cite: 79, 80].

Ollama Download

Modell laden (in CMD):

ollama pull phi3.5

Netzwerk-Bypass (Wichtig): Ollama beenden & neu starten:

set OLLAMA_HOST=0.0.0.0 && ollama serve
