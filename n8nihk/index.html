1. Infrastruktur (IHK-sicher via Docker)

Da lokale PowerShell-Skripte oft blockiert sind, nutzen wir Docker. Das isoliert n8n in einer eigenen sicheren Umgebung.

Schritt A: Docker Desktop installieren

Laden Sie Docker Desktop herunter und starten Sie den Rechner nach der Installation neu.

Schritt B: n8n in Docker starten

Kopieren Sie diesen Befehl in Ihre CMD (Eingabeaufforderung), um n8n zu starten:

docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n

Danach erreichbar unter: http://localhost:5678

2. Lokale KI (Ollama & Phi-3)

Wir betreiben unsere eigene KI lokal auf dem Rechner – ohne Cloud, ohne Datenabfluss.

Schritt A: Ollama installieren
Schritt B: Modell laden & Netzwerk öffnen

Geben Sie nacheinander diese Befehle in Ihre CMD ein:

ollama pull phi3

Damit Docker mit Ollama sprechen darf (Netzwerk-Bypass):

set OLLAMA_HOST=0.0.0.0 && ollama serve

Wichtig für n8n: Nutzen Sie als Base-URL im Ollama-Node
http://host.docker.internal:11434
