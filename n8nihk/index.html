1 KI-Basis: Ollama & Phi-3

Wir installieren die KI lokal auf Ihrem Rechner.

A) Download:
Ollama für Windows laden

B) Modell laden: Öffnen Sie die CMD (Eingabeaufforderung) und geben Sie ein:

ollama pull phi3

C) Netzwerk öffnen (WICHTIG): Beenden Sie Ollama (Taskleiste -> Quit) und starten Sie es in der CMD neu:

set OLLAMA_HOST=0.0.0.0 && ollama serve

2 n8n via Docker Desktop

Docker umgeht die Windows-Sperren für Skripte.

A) Installation:
Docker Desktop laden
⚠️ Wichtig: Nach der Installation Docker Desktop im Startmenü suchen und manuell starten. Warten Sie, bis der kleine Wal unten rechts grün leuchtet.

B) n8n starten: Wenn der Wal grün ist, öffnen Sie eine neue CMD und geben Sie ein:

docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n

Öffnen Sie danach: http://localhost:5678

3 Die n8n-KI Verbindung

Nutzen Sie im n8n Ollama-Node diese speziellen Einstellungen:

  • Base URL: http://host.docker.internal:11434
  • Model: phi3
