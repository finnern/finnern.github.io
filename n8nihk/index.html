📂 Zu den Workshop-Slides

⚡ 1. Den Kessel anfeuern (n8n & Docker)

Nutzen Sie die CMD (Windows + R -> cmd):

docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n

Portal öffnen: http://localhost:5678

🧠 2. Lokale Intelligenz (Ollama)

Laden Sie das Modell Phi-3.5 in Ihre CMD:

ollama pull phi3.5

Netzwerk-Bypass für Docker aktivieren:

set OLLAMA_HOST=0.0.0.0 && ollama serve
