1. n8n Starten (Der stabile Docker-Weg)

Wir nutzen Docker, um die Windows-Skript-Sperren zu umgehen.

Schritt A: Terminal öffnen

Drücken Sie Windows-Taste + R, geben Sie cmd ein und drücken Sie Enter.

Schritt B: n8n Befehl ausführen

Kopieren Sie diesen Befehl in die CMD:

docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
Fehler "Conflict"? Falls der Name schon belegt ist, nutzen Sie erst diesen Befehl zum Löschen:
docker rm -f n8n

Login unter: http://localhost:5678

2. Lokale KI (Ollama & Phi-3)

Modell laden

Geben Sie diesen Befehl in Ihre CMD ein:

ollama pull phi3
Netzwerk für Docker öffnen (Wichtig!)

Damit n8n die KI findet, beenden Sie Ollama (Taskleiste) und starten es so neu:

set OLLAMA_HOST=0.0.0.0 && ollama serve

3. Den n8n Account einrichten

Beim ersten Aufruf müssen Sie einen "Owner Account" erstellen:

  • Email: Dient als Benutzername (muss keine echte Mail sein).
  • Password: Mindestens 8 Zeichen, 1 Zahl und 1 Großbuchstabe.

Daten werden nur lokal auf diesem Rechner gespeichert.

4. Workshop Workflows
