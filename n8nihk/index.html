1. n8n Starten (Windows + R -> CMD)

Wir nutzen Docker, um Windows-Sicherheitssperren zu umgehen.

  1. Drücken Sie Windows-Taste + R, geben Sie cmd ein und drücken Sie Enter.
  2. Kopieren Sie diesen Befehl in die schwarze Box (CMD):
docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
Fehler "Conflict"? Falls n8n schon existiert, erst löschen:
docker rm -f n8n

Danach im Browser öffnen: http://localhost:5678

2. Lokale KI (Ollama & Phi-3.5)

Geben Sie diese Befehle nacheinander in die CMD ein:

Schritt A: Modell herunterladen ollama pull phi3.5 Schritt B: Netzwerk für Docker öffnen

Beenden Sie Ollama (Taskleiste -> Quit) und starten Sie es so neu:

set OLLAMA_HOST=0.0.0.0 && ollama serve

3. n8n Account einrichten

Erstellen Sie Ihren lokalen Account (Daten bleiben auf diesem PC):

  • Email: Frei wählbar (z.B. test@ihk.de)
  • Passwort: Mind. 8 Zeichen, 1 Zahl, 1 Großbuchstabe

4. Mission: Erwecke "IhKE" zum Leben 🤖

IhKE ist dein persönlicher IHK-Assistent. Kopiere den Code und füge ihn in n8n ein:

Import-Anleitung:
  1. Klicke oben auf den Button.
  2. Gehe in dein n8n Dashboard und drücke Strg + V.
  3. Klicke auf den Ollama Chat Model Node.
  4. Wähle unter "Credential" -> "Create New".
  5. Base URL: http://host.docker.internal:11434
